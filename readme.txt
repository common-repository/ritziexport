== Description ==
This plugin allow users to import catalog to the Ritzi App.

== Installation ==
1. Upload the plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Go to plugin settings and click \'Export products\'

== Frequently Asked Questions ==

= How to create a XML feed? =
Go to plugin settings and click 'Export products'
